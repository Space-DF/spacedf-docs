# SpaceDF Release v0.0.2 

This release introduces **Geofencing** and **Automation capabilities**, along with API improvements and internal refactoring to enhance system reliability and scalability.

---

## New Features

### Geofencing Engine
Core support for location-based logic and geofence management.

- Implemented geofencing core logic and data model  
- Added **Geofence Testing API** to simulate and validate triggers 

### Automation Engine
Initial release of the automation system.

- Implemented automation workflows for event-driven actions  

### API & Developer Experience

- Integrated **Swagger UI** for API documentation and testing  
- Added endpoint to retrieve **device list without pagination** 

---

## Improvements

- Refactored **device model** for better structure and maintainability  
- Improved **pagination mechanism** for more consistent API responses  

---

## Bug Fixes

- Fixed issue where **device cache was not cleared after deletion**  
---

## Maintenance

- General internal improvements and codebase cleanup
