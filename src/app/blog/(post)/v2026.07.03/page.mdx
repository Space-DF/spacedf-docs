# SpaceDF Release - v0.0.6

This release introduces the first version of **White Label Studio**, enabling organizations to customize their own branded experience through brand customization and custom domain management. It also includes significant frontend and backend performance optimizations, together with various bug fixes and stability improvements.

---

## New Features

### White Label Studio

Introduced a brand-new **White Label Studio** that allows organizations to build and manage their own branded experience.

#### Brand Customization

- Support centralized **Brand Identity** management.
- Configure organization branding, including:
  - Logo
  - Browser Favicon
  - Authentication Page
  - Email Branding
- Automatically generate Light and Dark logo variants from a single uploaded logo.
- Automatically generate browser favicon with optional manual override through Advanced Logo Settings.
- Improved branding workflow with reusable brand variables.

#### Custom Domain

- Support configuring a custom domain for published applications.
- Improved publishing workflow with dedicated **Publish** and **Update** actions.
- Support opening the published branded site directly after the first successful publish.

---

## Improvements

### White Label Experience

- Simplified logo management by reducing multiple upload points into a single upload flow.
- Improved White Label configuration consistency across the platform.
- Enhanced branding configuration workflow for a more intuitive user experience.
- Improved publish experience for managing branded applications.

### Platform Performance

- Optimized frontend rendering performance.
- Optimized backend processing and API performance.
- Improved overall application responsiveness.
- Reduced unnecessary frontend processing.
- Improved platform scalability and maintainability.
