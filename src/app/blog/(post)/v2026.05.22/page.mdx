# SpaceDF Release - v0.0.5

This release introduces the complete Smart Building flow, including improvements to Digital Twin interactions, device placement, bulk device onboarding, and notification enhancements. It also includes Device Direction support and performance optimizations for a smoother Smart Building experience.

---

## New Features

### Complete Smart Building Flow

- Support adding devices to Smart Building via sidebar
- Allow displaying device entities directly on the 3D model
- Support managing entity visibility through Device Details

### Bulk CSV Device Import

- Improved bulk CSV import flow for multiple device models
- Support assigning device models after CSV upload
- Added bulk actions for device model assignment
- Improved import visibility and tracking for uploaded devices

### Device Direction

- Added Device Direction feature to support device movement handling
- Added direction validation before movement execution

### Notification Browser

- Added Notification Browser support
- Support viewing and monitoring notifications within the platform

## Improvements
### Smart Building Performance

- Improved first loading performance 
- Optimized 3D model switching performance for smoother navigation between models
- Simplified Smart Building device flow to reduce UI complexity and improve usability
- Removed device adding action directly on the 3D model

### User Experience

- Improved bulk import UX for better device model assignment clarity
- Enhanced onboarding experience with contextual Smart Building guidance
- Updated Smart Building flow for simpler and more intuitive interaction

## Bug Fixes

- General fixes & Improved Smart Building interaction stability
- General fixes and improvements for notification and device management flow

## Maintenance

- General internal refactoring and system cleanup
- Performance optimization for Smart Building rendering and device interaction
- Improved maintainability for Smart Building-related modules
