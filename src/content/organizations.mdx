# Organizations & Users

Learn how to manage multi-tenant organizations and user access in SpaceDF.

## Organization Management

Organizations provide data isolation and multi-tenancy in SpaceDF.

### Creating Organizations

Organizations can be created through:
- Web console interface
- Management commands
- API endpoints

### Organization Settings

Configure your organization:
- Organization name and branding
- User access policies
- Integration settings
- Billing configuration

## User Management

### User Roles

SpaceDF supports multiple user roles:

- **Admin**: Full access to organization settings
- **Manager**: Device and space management
- **Viewer**: Read-only access to data
- **Custom**: Configurable permissions

### Inviting Users

Add team members to your organization:
1. Navigate to User Management
2. Send email invitations
3. Configure user roles and permissions

## White-Label Features

Customize the platform for your brand:
- Custom logos and colors
- Domain configuration
- Email templates
- API branding
