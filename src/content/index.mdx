# Introduction

SpaceDF is a comprehensive **IoT platform** that supports multiple protocols and device types for **asset tracking, monitoring, and management**. Whether you're building fleet tracking systems, smart buildings, or smart city solutions, SpaceDF provides the tools and templates you need.

## Supported Protocols

Connect your devices using any of these protocols through our **Multi-Protocol Adapter (MPA) Service**:

<Cards>
  <Cards.Card title="LoRaWAN" href="/protocols/lorawan">
    **Long Range Wide Area Network** - Low-power, wide-area networking protocol for IoT devices with long battery life.
  </Cards.Card>
  <Cards.Card title="HTTP/HTTPS" href="/protocols/http">
    **REST API** - Standard web protocols for device communication and data exchange.
  </Cards.Card>
  <Cards.Card title="MQTT" href="/protocols/mqtt">
    **Pub/Sub Messaging** - Lightweight messaging protocol ideal for IoT device communication.
  </Cards.Card>
  <Cards.Card title="WebSocket" href="/protocols/websocket">
    **Real-time Data** - Full-duplex communication for real-time IoT applications.
  </Cards.Card>
  <Cards.Card title="Socket.IO" href="/protocols/socketio">
    **Real-time Engine** - Enhanced WebSocket library with fallbacks and auto-reconnection.
  </Cards.Card>
  <Cards.Card title="CoAP" href="/protocols/coap">
    **Constrained Application** - Specialized protocol for resource-constrained devices.
  </Cards.Card>
  <Cards.Card title="LwM2M" href="/protocols/lwm2m">
    **Device Management** - Lightweight protocol for device management and service enablement.
  </Cards.Card>
  <Cards.Card title="SNMP" href="/protocols/snmp">
    **Network Management** - Standard protocol for managing and monitoring network devices.
  </Cards.Card>
</Cards>
