# Spaces & Location

Manage location-based groupings and tracking in SpaceDF.

## What are Spaces?

Spaces represent physical areas where your devices operate:
- Buildings and floors
- Outdoor areas and zones
- Virtual boundaries
- Geographic regions

## Creating Spaces

### Space Configuration

Set up spaces in your organization:

1. **Basic Information**: Name, description, and type
2. **Geographic Bounds**: Define the physical area
3. **Location Settings**: Configure positioning parameters
4. **Device Assignment**: Assign devices to spaces

### Space Types

Different space configurations:
- **Indoor Spaces**: Buildings, rooms, warehouses
- **Outdoor Spaces**: Parking lots, campuses, fields
- **Mixed Environments**: Indoor/outdoor combinations

## Location Tracking

### RSSI Trilateration

SpaceDF uses RSSI-based positioning:
- Multiple gateway measurements
- Triangulation algorithms
- Real-time position updates
- Accuracy optimization

### Location Services

Available positioning features:
- Real-time tracking
- Location history
- Movement analytics
- Geofencing alerts
