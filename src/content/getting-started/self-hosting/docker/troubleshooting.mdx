### Cannot installation SpaceDF
<h4 className="font-semibold text-xl my-3">Symptoms</h4>

- Terminal shows errors while installing or starting SpaceDF
- **RabbitMQ container fails** to start or keeps restarting
- Logs show authentication or permission errors related to RabbitMQ

<h4 className="font-semibold text-xl my-3">Common causes</h4>
- A **RabbitMQ service is already running** on the host machine.
- Existing RabbitMQ containers were created with **different credentials**.
- `RABBITMQ_DEFAULT_USER` or `RABBITMQ_DEFAULT_PASS` conflicts with:
    - A previous SpaceDF installation
    - Another project using RabbitMQ on the same machine
- Another project using RabbitMQ on the same machine
